## ⭐ 特性

1. 自定捕获当前页面收藏（红心）的推文信息

2. 支持将收藏的推文批量导出至剪切板或批量下载
   
3. 支持多种筛选、搜索策略
   
4. 支持自定义触发代码

  可以实现：Eagle 自动收藏、同步到其它平台等。
